--- layout: default title: QUAIL ---

QUAIL is a jQuery and Sizzle-comptable library for checking content against accessibility guidelines. It provides a flexible way to test for certain problems (say, images missing an alt text) and a collection of over 250 tests to get you started.

QUAIL is in the process of undergoing a big move from a hunble and unweildy PHP library to a lightweight and powerful accessibility checker. You can still find the 1.0 version of QUAIL, or join us to finish up the new version.
